This repo provides a complete implementation for replicating the paper "Simulation of Health Time Series with Nonstationarity" by Toye et al. (2024).

CS598_DLH_Project/
├── eda/                    # EDA simulation functions
├── glucose/                # Glucose simulation functions  
├── ohio/                   # OHIO-T1DM dataset for glucose
└── wesad/                  # WESAD dataset for EDA/stress
